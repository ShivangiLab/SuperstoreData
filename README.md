Tableau Dashboard – Profitability Analysis
# Objective
To analyze product profitability and support decision-making using interactive dashboards.

# Tools Used
- Tableau
- Excel

# Key Work
- Designed an **interactive dashboard** with multiple visualizations  
- Implemented **filters and drill-down features** for user interaction  
- Analyzed **least profitable items across categories and regions**  
- Used visual storytelling to communicate insights effectively  

# Dashboard Features
- Profit by category (bar chart)  
- Top 10 least profitable items  
- Profit by state (map)  
- Profit trend over time (line chart)
